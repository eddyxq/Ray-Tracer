About
Operating System: Fedora 28 (x86-64)

Compiler: g++ 8.1.1

Used template code provided on D2L.

Compiling and Running
Execute the run.sh file to compile and run the program.

$ bash run.sh
Alternatively, execute the makefile to compile,

$ make
and run the program with:

$ ./Boilerplate.out

Controls

Note: You must re-run the program after each render.

Enter the scene number you wish to view
1 - scene 1
2 - scene 2
3 - scene 3

Enter a focal length adjustment
0 - default
positive - e.g. 100 to zoom in
negative - e.g. -100 to zoom out
